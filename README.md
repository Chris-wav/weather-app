# 🌦️ Weather App

A simple weather forecast app using the OpenWeatherMap API. Users can enter a city name and get real-time weather info like temperature, humidity, and weather icons.

## 🔧 Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- OpenWeatherMap API
- Fetch API
